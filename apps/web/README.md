# Web App Notes

Recommended stack: Next.js + TypeScript + Tailwind.

Suggested routes:
- `/` dashboard
- `/reports`
- `/reports/new`
- `/reports/[id]`
- `/templates`
- `/themes`
- `/validation`
- `/exports`

Key screens:
- upload / create report
- JSON preview / validation panel
- section preview
- export settings / PDF generation
