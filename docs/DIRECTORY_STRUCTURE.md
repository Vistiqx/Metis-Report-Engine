# Directory Structure

`D:\__Projects\Metis-Report-Engine`

- `schema/` JSON Schema contracts
- `engine/` parsing, validation, scoring, rendering, visuals
- `templates/` Jinja templates and report components
- `theme/` design tokens and client profiles
- `prompts/` AI prompts for autonomous build and structured outputs
- `examples/` sample payloads
- `tests/` contract and rendering tests
- `docs/` implementation guidance
