# UI Guidelines

Use the same design language as Metis where practical.

## Layout
- Left sidebar navigation
- Main content workspace
- Right panel optional for validation / JSON preview / export settings

## Sections
- Dashboard
- Reports
- Templates
- Themes
- Visualizations
- Validation
- Exports
- Settings

## Brand palette
- Tropical Indigo `#AC8FFE`
- Amethyst `#8A6FD6`
- Ultra Violet `#4C3D75`
- Night `#121212`
- Lavender `#E8E8F8`
- Dim Gray `#6E6E80`

## Typography guidance
- Headings: high-contrast serif or strong display pairing
- Body: clean sans-serif
- Monospace: for IDs, evidence hashes, version fields

## UX notes
- Keep the app feeling like an operator console / mission control.
- Validation warnings should be visible but non-destructive.
- PDF preview should feel executive-grade, not developer-centric.
